HTML Elements

What is HTML?

HyperText Markup Language

What are HTML elements?

HTML elements are basically the building blocks of any web application that is meant to be viewable to users. Elements are designated by tags, often including an attribute and its value.

What are HTML tags?

The HTML "code" for a specfic HTML element. They look like this:

<element> content </element>

Examples:

  • <p> </p> = paragraph; used for textual information.
  • <body> </body> = body; encapsulates the entire web page
  • <a> </a> = link; defines a link to another web page
  • <ol> </ol> = ordered list; encapsulates a numbered list of items
  • <h1> </h1> = header, level 1; the highest (largest) heading, often used for titles

You'll notice that all of these tags have an opening tag (<element>) and a closing tag (</element>).

However, some tags only have an opening tag, such as:

<img> = image; used to designate pictures. For this specific tag to work, it needs it's href attribute to define the image file.

What are HTML properties?

Properties are defined by the Document Object Model (DOM).

Although most properties are strings, some can be objects.

What are attributes?

They customize the tag in different ways. The CSS selectors "class" and "id" are both attributes.

<tag attribute="value"> content </>

Example: Image

<img href="https://www.example.com"> click this link </>

Attributes are can be both upper or lowercase; it doesn't actually matter.

Attribute values are always strings, not actual numbers.

What are values?

Values refer to the type of attribute being used.

What are the different types of HTML elements?

  • Main root: <html> is thetop-level element and can only be used once. All other elements are descendants of the root element.
  • Document metadata: information about the webpage such as styles, scripts, and data to help the software display the webpage.
  • Sectioning root: <body> is the only one used to encapsulate the content of the HTML document.
  • Content sectioning: These elements logically organize the document's content in an outline format. Examples: header, main, article, section, section headings, etc.
  • Text content: These elements further organize content sectioning elements. Examples: block quotes, ordered and unordered lists, list items, paragraphs, etc.
  • Inline text semantics: These elements specify the structure and style of a piece of text. Examples: hyperlinks, abbreviations, line break, boldface text, emphasized text, etc.
  • Image and Multimedia: These elements support various multimedia resources. Examples: images, videos, audio, etc.
  • Embedded content: These elements support other types of embedded content. Examples: <embed>, <object>, <portal>, etc.
  • SVG and MathML: Used to embed SVG and MathML content
  • Scripting: Use to allow dynamic content by supporting scripting languages like JavaScript.
  • Demarcating edits: Lets users know some content has been specifically altered. Example: deletions and insertions
  • Table content: Best used to express tabular data; any type of information that is best conveyed in a more visual format than purely textual.
  • Forms: Used to create forms that the user can input and submit information to the website or application. Examples: buttons, text fields, selectable options, labels, etc.
  • Interactive elements: Allows other types of interactive content like disclosure widgets and dialog boxes.
  • Web components: Customizable elements to perform unique functions. Examples: <slot> and <template>
  • Obsolete and deprecated elements: Older elements that are no longer compatible with the newest version of HTML.
