HTML Elements

What is HTML?

HyperText Markup Language

What are HTML elements?

HTML elements are basically the building blocks of any web application that is meant to be viewable to users. Elements are designated by tags, often including an attribute and its value.

What are HTML tags?

The HTML "code" for a specfic HTML element. They look like this:

<element> content </element>

Examples:

  • <p> </p> = paragraph; used for textual information.
  • <body> </body> = body; encapsulates the entire web page
  • <a> </a> = link; defines a link to another web page
  • <ol> </ol> = ordered list; encapsulates a numbered list of items
  • <h1> </h1> = header, level 1; the highest (largest) heading, often used for titles

You'll notice that all of these tags have an opening tag (<element>) and a closing tag (</element>).

However, some tags only have an opening tag, such as:

<img> = image; used to designate pictures. For this specific tag to work, it needs it's href attribute to define the image file.

What are HTML properties?

Properties are defined by the Document Object Model (DOM).

Although most properties are strings, some can be objects.

What are attributes?

They customize the tag in different ways. The CSS selectors "class" and "id" are both attributes.

<tag attribute="value"> content </>

Example: Image

<img href="https://www.example.com"> click this link </>

Attributes are can be both upper or lowercase; it doesn't actually matter.

Attribute values are always strings, not actual numbers.

What are values?

Values refer to the type of attribute being used.
