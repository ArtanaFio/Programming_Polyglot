The position property is used to place HTML elements in different ways.

There are five possible values for position that change the behavior of the element:

  • Static
  • Relative
  • Fixed
  • Absolute
  • Sticky

Static Positioning

CSS selector {
position: static;}

Static is the default position setting for all elements. When set to static, the position will not be affected by the top, right, bottom, and left properties.

Relative Positioning

CSS selector {
position: relative;}

The same as static, except relative positioning is affects by the forementioned properties. Those properties will displace the HTML element relative to its normal position in the document.

Fixed Positioning

CSS selector {
position: fixed;}

Fix positioning allows the developer to position the HTML element relative to the viewport such that the element remains on screen as the user scrolls. This removes the element from the normal flow of the document and is prefered for navigation bars, floating chat buttons and chat boxes. The top, right, bottom, and left properties can be used to position the element within the viewport.

This web page's sidebar uses a fixed position and the code to set it up is displayed below:

<style> aside, div { width: 49vmin; } aside div { position: fixed; align-items: center; } aside div ul { margin: 0; margin-top: 5vmax; padding: 0 3vmin; list-style-type: none; line-height: 0; } aside div ul li { padding: 3vmin; text-align: center; } aside div ul li a { text-decoration: none; color: white; padding: 1vmin 3vmin; } aside div ul li a:hover { font-weight: bold; background-color: rgba(102, 248, 248, 0.603); border-radius: 10px; box-shadow: inset 1vmin 1vmin 2vmin teal, inset -1vmin -1vmin 2vmin teal, inset 1vmin -1vmin 2vmin teal, inset -1vmin 1vmin 2vmin teal; width: -moz-fit-content; width: fit-content; }
</style>

Absolute Positioning

CSS selector {
position: absolute;}

Absolute positioning lets the developer to position the HTML element at a specific point without affecting other elements and not being affected by them. It removes the element from the normal document flow. The top, right, bottom, and left properties can be used to position the HTML element.

Suggest elements for absolute positioning:

  • modals
  • images with captions
  • icons on top of elements

In other cases, especially entire page layouts, flexbox or grid is preferred over absolute positioning.

Sticky Positioning

CSS selector {
position: sticky;}

Sticky is a cross between static and fixed positioning because the HTML element will act like a normal static element until the user scrolls past the element's original position. Once past that point, the element behaves like a fixed element and remains positioned relative to the viewport until the user scrolls back to the element's original position. This is useful for section headings such as categories on a shopping app or the header cells in a large table or chart.
