CSS selectors act as a sort of label for identifying specific HTML elements to apply style to them. There are different types of CSS.

What are the different kinds of CSS selectors?

There are common selectors that select elements based on their type, class, and ID.

Relatiionships between HTML elements are used by child selectors and sibling selectors to select HTML elements.

Attribute selectors target HTML elements that have a specific attribute or value.

What are the common selectors?

Common selectors are the most often used because they are the simplest to understand and apply.

  • Type selector targets an element by it's tag name.
  • Class selector assigns a class attribute to the element and is denoted by ".class-name". Classes are useful when you want to apply the same style to multiple HTML elements and when you want to apply more than one set of styles to an element.
  • ID selector assigns an ID attribute to the element and is denoted by "#ID-name". IDs are best used for elements that have very unique styles that are specific to that one HTML element because you can only define one Id for each element and cannot reuse that ID's name.
Common Selector HTML CSS
Type <h1> h1 {
font-size: 5rem;
}
Class <p class="color"> .color {
color: 'cyan';
}
ID <p id="first-line"> #first-line {
font-weight: bold;
}

What are child selectors?

This selector type targets elements based on their relationship to other elements, specifically the parent-child relationship.

  • Descendant selector not only does it select the child of the parent element, it selects all descendants of that parent element.
  • Direct child selector only targets direct child elements of the parent element.

What are sibling selectors?

This is the other selector that targets elements based on relationships. Sibling selectors select elements that have the same parent element.

  • General sibling selector will select all of the same sibling that come after the specified element. Using the family metaphor, it select all younger siblings that are identical twins.
  • Adjacent sibling selector is more specific and only selects the next closest sibling, regardless if it has a twin.

What are attribute selectors?

These selectors use the HTML element's attributes and their values to apply CSS styling. All attribute selectors are encased in square brackets.

  • Attribute present selector selects the HTML elements that have the specified attribute even if there is no value.
  • Attribute equals selector selects the HTML elements that have the specified value for the attribute.
  • Attribute contains selector selects the HTML elements that contains a specified string in the attribute's value.
  • Attribute begins with selector selects the HTML elements that begins with a specified string in the attribute's value.
  • Attribute ends with selector selects the HTML elements that ends with a specified string in the attribute's value.
  • Attribute spaced selector selects the HTML elements that have a space in the attribute's value. Only one of the terms must match for the selection to work.
  • Attribute hyphenated selector selects the HTML elements that have attributes with hyphenated values. The first half of the hyphenated value must be stated for the selection to work.

What are pseudo-clases?

Similar to classes, except pseudo-classes select HTML elements based on their structure or position within the DOM tree. Styles are applied to these elements depending on how the user interacts with the element, link states, etc. They are preceeded by a colon.

  • Link pseudo-classes only has two regarding links: :link and :visited. These pseudo-classes change the link's style based on if it has been visited or not.
  • User action pseudo-classes apply styles to elements based on how the user interacts with them. :hover is applied when the user hovers the mouse over the element. :active applies style to an element when the user clicks on it. :focus applies style to an element when it becomes the focus point of the page.
  • User interface state pseudo-classes applies styles to the user interface state of elements, particularly form elements. :enabled targets an input when it's enabled (its default state). :disabled targets a disabled element. The :checked pseudo-class is used on checkboxes and radio buttons when they have been checked. The :indeterminate pseudo-class is used on the indeterminate state (neither checked nor unchecked) states of these elements.
  • Structural and position pseudo-classes includes many options for targeting HTML elements because these pseudo-classes are based on the location of the elements within the DOM tree. These pseudo-classes can select the first, last, and the only child of the parent element.
  • The Target pseudo-class styles HTML elements when their ID matches the URI fragment identifier(#).
  • The Empty pseudo-class selects the HTML element that doesn't have any children or text nodes: the empty elements.
  • Negation pseudo-class selects an element not represented by the stated value.

What are pseudo-elements?

They are dynamic elements that are created when specified in the CSS code. Only one pseudo-element can be used within a selector at a time.

  • Textual pseudo-elements target specific textual content. :first-letter selects the first letter of text inside the HTML element. :first-line identifies and selects the first line of text inside the element.
  • Generated pseudo-elements allow the developer to add more content or user interface components to the page without overcrowding the document with unsemantic elements. Both :before and :after create inline level pseudo-elements inside the selected element. These are commonly used with the content property to add nonimportant information. The :before pseudo-element is placed before the selected element while the :after pseud-element is placed after it.
  • The Fragment pseudo-element is ::selection, which selects the part of the HTML document that has been selected or highlighted by the user. Not all browswers support this pseudo-element, so the ::-moz-selection was created to work with Firefox. Keep in mind this pseudo-element can only be styled with the color, background, background-color, and text-shadow CSS properties.
